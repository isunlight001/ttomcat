# ttomcat

#### 项目介绍
写一个简略版tomcat，你也可以掌握tomcat原理。

你可以写一个简单的servlet项目，部署在对应的文件夹下就可以运行了。

类的命令和Tomcat源码保持一致，方便你去理解...
